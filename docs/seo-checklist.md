# SEO checklist перед публикацией

## Домен и базовые файлы

- [ ] Заменить `https://your-domain.ru` на реальный домен в `src/data/site.ts`.
- [ ] Заменить домен в `public/robots.txt`.
- [ ] Заменить домен в `public/sitemap.xml`.
- [ ] Обновить даты `lastmod` в `sitemap.xml`.
- [ ] Проверить, что `/robots.txt` открывается.
- [ ] Проверить, что `/sitemap.xml` открывается.
- [ ] Проверить, что `/favicon.svg` открывается.
- [ ] Проверить, что `/og-image.svg` открывается.

## Яндекс.Вебмастер

- [ ] Добавить сайт в Яндекс.Вебмастер.
- [ ] Получить verification meta.
- [ ] Добавить verification в `site.yandexVerification`.
- [ ] Отправить `sitemap.xml` в Яндекс.Вебмастер.
- [ ] Проверить индексацию главной страницы.
- [ ] Проверить отсутствие критических ошибок обхода.

## Яндекс.Метрика

- [ ] Создать счётчик Яндекс.Метрики.
- [ ] Заменить `PUBLIC_YANDEX_METRIKA_ID=0` на реальный ID.
- [ ] Проверить, что счётчик загружается.
- [ ] Проверить цели кликов по CTA.
- [ ] Проверить цели отправки форм.
- [ ] Проверить цели просмотра страниц.

## Meta tags

- [ ] На каждой странице есть один `<title>`.
- [ ] На каждой странице есть `<meta name="description">`.
- [ ] На каждой странице есть `<link rel="canonical">`.
- [ ] На важных страницах нет `noindex`.
- [ ] Проверить Open Graph:
  - [ ] `og:title`;
  - [ ] `og:description`;
  - [ ] `og:image`;
  - [ ] `og:url`.
- [ ] Проверить Twitter Card.

## Schema.org

- [ ] На главной есть WebSite schema.
- [ ] На главной есть Person schema.
- [ ] На страницах услуг есть Service schema.
- [ ] На кейсе есть CreativeWork schema.
- [ ] BreadcrumbList есть на ключевых внутренних страницах.
- [ ] FAQPage есть на страницах с FAQ.

## Контент

- [ ] На каждой странице один H1.
- [ ] Заголовки не дублируются без необходимости.
- [ ] Формулировки не сводятся к “делаю сайты”.
- [ ] Основной акцент: автоматизация процессов и цифровые системы.
- [ ] Проверить орфографию и пунктуацию.
- [ ] Проверить телефон.
- [ ] Проверить ссылки CTA.

## Техническая проверка

- [ ] `npm run build` проходит без ошибок.
- [ ] Нет ошибок в Console.
- [ ] Нет горизонтального скролла на мобильном.
- [ ] Формы работают.
- [ ] Backend fallback работает, если API не указан.
- [ ] Реальная отправка работает, если API указан.
- [ ] Проверить скорость загрузки.
- [ ] Проверить адаптивность:
  - [ ] 360px;
  - [ ] 390px;
  - [ ] 768px;
  - [ ] 1024px;
  - [ ] 1280px;
  - [ ] 1440px.
