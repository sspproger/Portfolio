# Production checklist

## Контент

- [ ] Проверены все заголовки.
- [ ] На каждой странице один H1.
- [ ] Нет формулировок “делаю сайты быстро и недорого”.
- [ ] Основной фокус — автоматизация процессов.
- [ ] Проверен телефон.
- [ ] Проверены все CTA.
- [ ] Проверены тексты форм.

## Адаптивность

- [ ] 320px.
- [ ] 360px.
- [ ] 390px.
- [ ] 768px.
- [ ] 1024px.
- [ ] 1280px.
- [ ] 1440px.
- [ ] Нет горизонтального скролла.
- [ ] Мобильное меню работает.
- [ ] Формы не вылезают за экран.
- [ ] Кнопки корректно переносятся.

## SEO

- [ ] Заменён домен в `src/data/site.ts`.
- [ ] Заменён домен в `public/robots.txt`.
- [ ] Заменён домен в `public/sitemap.xml`.
- [ ] Проверен `<title>`.
- [ ] Проверен `<meta name="description">`.
- [ ] Проверен canonical.
- [ ] Проверен Open Graph.
- [ ] Проверен Twitter Card.
- [ ] Проверена Schema.org-микроразметка.
- [ ] Сайт добавлен в Яндекс.Вебмастер.
- [ ] Sitemap отправлен в Яндекс.Вебмастер.

## Аналитика

- [ ] Создан счётчик Яндекс.Метрики.
- [ ] Указан `PUBLIC_YANDEX_METRIKA_ID`.
- [ ] Проверены цели CTA.
- [ ] Проверены цели форм.
- [ ] Проверены цели page view.
- [ ] Проверены UTM-метки.

## Формы

- [ ] Demo fallback работает без API.
- [ ] `PUBLIC_API_BASE_URL` задан на production.
- [ ] Заявка уходит на backend.
- [ ] Telegram-уведомление приходит.
- [ ] Success-состояние работает.
- [ ] Error-состояние работает.
- [ ] Поля не очищаются при ошибке.
- [ ] UTM сохраняются.

## Backend

- [ ] Cloud Functions созданы.
- [ ] API Gateway создан.
- [ ] Function IDs заменены в `api-gateway.yaml`.
- [ ] `TELEGRAM_BOT_TOKEN` задан.
- [ ] `TELEGRAM_CHAT_ID` задан.
- [ ] `ALLOWED_ORIGIN` задан.
- [ ] CORS проверен.
- [ ] Логи Cloud Functions проверены.

## Безопасность

- [ ] `.env` не попадает в Git.
- [ ] Telegram token не хранится во frontend.
- [ ] CORS ограничен production-доменом.
- [ ] Есть server-side validation.
- [ ] Payload ограничен по размеру.

## Performance

- [ ] Нет тяжёлых изображений.
- [ ] Скриншоты лениво загружаются.
- [ ] Нет лишних JS-библиотек.
- [ ] `npm run build` проходит.
- [ ] `npm run preview` работает.

## Accessibility

- [ ] Есть skip-link.
- [ ] Фокус виден.
- [ ] У форм есть label.
- [ ] У мобильного меню есть aria-expanded.
- [ ] У success/error есть aria-live.
- [ ] У изображений есть alt.
